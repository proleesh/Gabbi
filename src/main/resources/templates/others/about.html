About Us

우리는 Proleesh입니다. 현재 팀원은 1명뿐이지만, 새로운 기술과 미래를 꿈꾸며 나아가고 있습니다. 혁신적인 아이디어와 기술을 통해 세상을 변화시키는 것을 목표로 하고 있으며, 그 여정은 이제 막 시작되었습니다. Proleesh는 끊임없는 도전과 성장을 통해 더 나은 내일을 만들어가는 것을 지향합니다. 혼자서도 시작할 수 있다는 믿음과 열정으로, 우리는 기술 혁신을 선도하며 미래를 개척해 나갈 것입니다. 앞으로 더 많은 팀원들과 함께 성장하며, 꿈꾸는 미래를 현실로 만들어 나가겠습니다.

현재 CTO(최고 기술 책임자)는 Sung-Hyuk Lee 입니다. 최신과 안정성 그리고 시큐리티을 추구하는 기술자입니다.

사용되는 주요 기술

Spring:

Spring 프레임워크는 Java 플랫폼을 위한 오픈 소스 애플리케이션 프레임워크로, 특히 웹 애플리케이션 개발을 쉽게 하기 위해 설계되었습니다. IoC (Inversion of Control)와 DI (Dependency Injection) 패턴을 통해 객체 간의 의존성을 효율적으로 관리할 수 있습니다. Spring Boot는 Spring 기반의 애플리케이션을 빠르게 구축하기 위한 도구로, 설정을 최소화하고 개발을 가속화합니다.

Java:

Java는 널리 사용되는 객체 지향 프로그래밍 언어로, 안정성과 플랫폼 독립성을 특징으로 합니다. JVM (Java Virtual Machine)을 통해 다양한 운영 체제에서 실행될 수 있으며, 강력한 표준 라이브러리와 풍부한 생태계를 제공합니다. 웹, 모바일, 데스크톱 애플리케이션 등 다양한 분야에서 활용됩니다.

Thymeleaf:

Thymeleaf는 Java 기반의 서버사이드 템플릿 엔진으로, HTML 파일을 템플릿으로 사용하여 동적인 웹 페이지를 생성합니다. Spring MVC와 잘 통합되어 있으며, 템플릿 파일을 그대로 브라우저에서 열어도 제대로 렌더링됩니다. 선언적 문법을 통해 간편하게 HTML을 조작할 수 있습니다.

MySQL:

MySQL은 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)으로, 신뢰성과 성능이 뛰어나며, 많은 웹 애플리케이션에서 널리 사용됩니다. SQL(Structured Query Language)을 사용하여 데이터를 저장, 검색, 수정할 수 있습니다. 대규모 트래픽을 처리할 수 있는 확장성과 안정성을 제공합니다.

MariaDB:

MariaDB는 MySQL의 포크(fork)로, 원래 MySQL의 개발자들이 주도하여 만들어졌습니다. MySQL과 호환성이 높으며, 성능과 기능을 향상시키는 다양한 개선 사항이 포함되어 있습니다. 오픈 소스 프로젝트로 활발히 개발되고 있으며, MySQL에서 쉽게 마이그레이션할 수 있습니다.

JavaScript (JS):

JavaScript는 웹 페이지를 동적으로 만들기 위해 사용되는 프로그래밍 언어로, 브라우저 내에서 실행됩니다. HTML과 CSS와 함께 웹 기술의 삼대 요소 중 하나로, 이벤트 처리, DOM 조작, AJAX 등을 통해 사용자 경험을 향상시킵니다. Node.js를 통해 서버 사이드 개발에서도 사용됩니다.

QueryDSL:

QueryDSL은 타입 안전한 쿼리를 작성할 수 있게 해주는 Java 라이브러리입니다. JPQL, SQL, MongoDB 등 다양한 데이터베이스와 ORM 툴과 함께 사용할 수 있으며, 코드 작성 시점에서 쿼리의 문법적 오류를 방지할 수 있도록 도와줍니다. 메타 모델을 생성하여 컴파일 시점에 쿼리를 검증합니다.
